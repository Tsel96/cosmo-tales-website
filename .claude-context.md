# Cosmo Tales Website — Session Context

## Project
Marketing website for "Cosmo Tales" — a retro-futuristic space game by Bohemia Interactive.
- **Stack**: React 19 + Vite 7 + Tailwind CSS v4 + React Router v7 (SPA mode)
- **Deployment**: Vercel (https://cosmo-tales-website.vercel.app)
- **Main file**: `app/routes/home.jsx`
- **Styles**: `app/app.css`

## Key Code Changes to Sync to Design

### 1. Feature Cards Layout
- Sections closer together: `gap-12` between video and text
- Videos bigger: `flex-[3]`
- Text thinner: `w-[280px] shrink-0`
- Max width: `max-w-[1200px]`
- Mobile: stack vertically (`flex-col`)

### 2. Typography
- Paragraph text uses Inter Display font (same as buttons) but regular weight
- OpenType features on body: `ss07, ss08, cv01, cv03, cv04, cv05, cv08, cv09, cv10`
- Letter spacing = 0 on paragraphs

### 3. 3D Tilt on Feature Videos
- Apple TV-style 3D tilt on hover (desktop only)
- Anti-flicker: flat parent div triggers mouse events, child div receives transform
- `perspective(800px) rotateY(±8deg) rotateX(±8deg)`
- Disabled on touch devices via `window.matchMedia('(hover: hover)')`

### 4. Footer
- Top row: links left-aligned (`self-start`) — Press Kit, Discord, Twitter
- Bottom row: BI logo (left, links to bohemia.net) + copyright text (right)
- Copyright: "© 2026 BOHEMIA INTERACTIVE a.s. Cosmo Tales® and BOHEMIA INTERACTIVE® are registered trademarks..."
- BI logo file: `public/bi-logo-white.svg`

### 5. Steam Wishlist Button
- Left padding `pl-1` (4px) on Steam icon for optical balance

### 6. Mobile Responsiveness
- All hover effects wrapped in `@media (hover: hover)` in CSS
- Hero: `h-[75vh]` on mobile (vs `h-[85vh]` desktop)
- Ship on mobile: `width: 56vh`, centered (`left: 50%, translate(-50%, -50%)`), `top: 58%` (close to text)
- Ship uses wrapper div for centering, inner img for levitate animation (prevents transform conflict)
- Hero visual container: `height: 75vh` on mobile, `width: 75vh`
- Feature cards stack vertically on mobile
- Email form stacks vertically on mobile
- Footer stacks vertically on mobile
- Font sizes smaller on mobile (e.g., h1: 36px mobile vs 56px desktop)
- Padding: `px-5` mobile vs `px-12` desktop

### 7. Animation Rules (Emil Kowalski's 8 Rules)
- All transitions ≤300ms
- Video zoom: 250ms with spring easing
- Input glow: spring easing
- Link reveal: 200ms spring easing
- Tilt leave: 280ms cubic-bezier(0.22, 1, 0.36, 1)
- Button press: scale(1.04) hover, scale(0.97) active, 80ms active duration
